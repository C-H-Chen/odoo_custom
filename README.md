# 📦 Odoo ERP 客製化功能模組

專為企業內部流程優化所開發的 Odoo ERP 客製化模組集合。涵蓋銷售業務、庫存採購、生產製造及自動化通知等核心領域，透過功能與介面客製化，大幅降低人工作業成本並提升跨部門協作效率。

## 🛠 技能與工具
- **後端**: Python (Odoo ORM, XMLRPC)
- **前端**: XML, CSS (QWeb Reports, Form/Tree/Kanban Views)
- **資料庫**: PostgreSQL
- **基礎架構與部署**: Linux, Docker
- **第三方服務串接**: LINE Notify API, Mail Server (SMTP)

---

## ✨ 核心模組功能概覽 

### 💼 銷售
- **自動化計價與折扣系統**：客製化報價單、銷售訂單 (SO) 及詢價單 (RFQ)，內建出貨價與未連稅計價邏輯，自動帶入價格與折扣，避免人工計算誤差。
- **歷史交易記憶功能**：設計「客戶歷史折扣與商品資料記錄」，建立訂單時自動帶出客戶最後一次折扣或專屬廠商料號，提升業務下單效率。
- **彈性包裝計價**：訂單模組支援依「包裝單位」自動轉換並計算價格，符合多樣化的批發/零售計價需求。
- **進階訂單總表檢視**：優化訂單總表 (Tree View) 介面，新增一鍵展開/收合產品明細功能，協助管理者快速掌握訂單全貌。
- **客戶消費軌跡追蹤**：開發網站訂單商品分類紀錄功能，精準追蹤並分析客戶購買習慣，作為後續行銷與銷售策略之數據基礎。

### 🔄 庫存與採購
- **多單合併出貨機制**：支援將多張銷售訂單商品合併轉入「單一出貨單 (DO)」，並具備自動結算與調整「未出貨數量」邏輯，降低物流與倉管作業成本。
- **採購連動**：製造訂單 (MO) 展開時，系統自動判定原物料/素材庫存水位，若低於安全庫存即自動生成對應的採購單 (PO)，確保生產不斷線。

### ⚙️ 製造
- **接單生產自動化 (MTO/BTO)**：建立銷售訂單時，系統自動核對現有庫存。若庫存不足，將直接於介面提供操作按鈕並自動生成「生產單 (MO)」，確保交期準確。
- **投料追蹤**：於製造模組客製化「未投料數量」欄位與運算邏輯，協助生管人員即時追蹤物料耗用狀態，防堵人為領退料錯誤。

### 📩 報表客製化與通知
- **跨平台即時通知 (Mail & LINE)**：串接 Mail Server 與 LINE Notify API，實現訂單成立、出貨狀態變更時的自動化推播，提升跨部門與客戶間的資訊同步率。
- **企業專屬單據列印**：客製化出貨單 (Delivery Slip) 顯示內容與專屬紙張列印格式 (QWeb Reports)，提升企業文件規範與品牌專業度。
